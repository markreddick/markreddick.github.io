Scry

$4.99

Quick reference to the D20 v3.5 SRD

Scry is a quick reference to the D20 v3.5 SRD (System Reference Document). This consists of races, classes, feats, skills, spells, magic items, monsters and psionic powers compatible with the d20 System version of Dungeons & Dragons (D&D) and other various role playing games from Wizards of the Coast.

With Scry, you can quickly drill down to the information you need. For instance, you can see a list of all the spells or just the spells for a particular class or drill down even further and get a list of spells for a particular level.

Features

d20 SRD 3.5

Scry gives you quick and easy access to the d20 version 3.5 System Reference Document (SRD).

This information includes races, classes, feats, skills, spells, magic items, monster and psionic powers compatible with the d20 System version of Dungeons & Dragons (D&D) and other various role playing games.

Unlike some other similar apps, Scry allows you to drill down in categories to get to more specific subsets of data instead of having to wade through a huge list of information. For instance, you can view a list of all the spells, or drill down to view all the spells for a particular class or drill down even further to view only the spells for a particular level for the selected class.

Pathfinder

Scry also gives you access to the Pathfinder RPG reference database (PRD).

This information includes classes, domains, equipment, feats, items, monsters, powers, races, skills, spells and traits.

You can switch between viewing the information for the d20 SRD and the Pathfinder reference database.

Bookmarks

You can create bookmarks for any page for quick access later.

Bookmarks can be organized into folders and sub-folders.

Search

All the lists (spell lists, item lists, monster lists, etc) now contain a search bar at the top which allows you to quickly locate specific items in the list.

Universal App

Scry is built as a Universal App which means it has been designed for both iPhone and iPad in one application. So, the one app will run natively on an iPhone, iPad or iPod touch. No need to pay separately for the "iPhone" and "iPad" versions like many other apps do.
